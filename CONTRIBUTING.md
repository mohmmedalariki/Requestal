# Contributing to Requestal

Thank you for exploring Requestal! 

## Getting Started

1.  **Install**: `npm install`
2.  **Dev**: `npm run dev` (Starts Vite server for HMR)
3.  **Build**: `npm run build` (Outputs to `dist/`)

## Architecture
-   **Side Panel**: Main UI lives in `src/panel/App.tsx`.
-   **Core Logic**: Network dispatch and conversion logic is in `src/utils/`.
-   **Manifest**: `public/manifest.json` defines the extension capability.

## Issues
Please report issues on the GitHub repository.
